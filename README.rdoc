== README

Steps are necessary to get theapplication up and running.


* Ruby version: ruby 2.1.5p273 (2014-11-13) [x86_64-linux-gnu]

* Rails version: Rails 4.1.8

* System dependencies: rails4-env in dockerhub by eduardosteps

* How to run: <tt> rails s </tt>

* Features: List events, organizations and Query Events based on organization or hostname order by timestamp

* Database Schema: 
 * Events Table ( message, organization, timestamp, hostname)
 * Organization Table ( name )
